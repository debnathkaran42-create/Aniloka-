Place manga PAGE images here using this folder structure:

images/manga/chapters/<manga-id>/<chapter-id>/<page-number>.jpg

Example, Chapter 1 of Manga 1 with 3 pages:
images/manga/chapters/manga-1/ch-1/1.jpg
images/manga/chapters/manga-1/ch-1/2.jpg
images/manga/chapters/manga-1/ch-1/3.jpg

Find the exact <manga-id> and <chapter-id> your project is expecting
by opening Admin Panel -> Manage Chapters -- the "path hint" box under
the chapter form shows the exact folder path to use.

If a page image is missing, the reader shows a placeholder box with
the expected file path instead, so you always know what to add.
